    UI/UX 设计原则分析：

    设计原则 说明 启示
    控制面分离 Gateway 是控制面，产品是助手 清晰的职责边界，UI 是控制界面而非完整产品
    多频道收件箱 统一界面管理所有频道消息 按频道分组的消息流，而非独立的聊天窗口
    实时状态 Presence（在线状态）、Typing Indicators（输入指示） 提升沟通体验，让用户感知对方状态
    渐进式增强 核心功能由 Gateway 提供，Apps 可选增强 基础功能稳定可用，高级功能通过 Apps 扩展
    安全优先 DM Pairing（配对码）、显式选择公开 DM 默认安全的隐私策略，需用户主动开放
    语音交互 Voice Wake（唤醒词）、Talk Mode（连续语音） 语音作为一等交互方式，而非附属功能
    视觉工作区 Canvas（画布）+ A2UI（Agent 驱动 UI） Agent 可以控制视觉界面，而不仅是文本
    极简主义 Gateway 后台运行，UI 仅显示必要信息 避免信息过载，聚焦于核心任务
    移动端原生 iOS/Android Nodes 提供原生体验 利用平台特性（推送、相机、位置等）

    2. 设计目标与原则¶

    2.1 核心设计目标¶

    目标 说明 验收指标
    能力不缩水 保留所有 18 项导航项，不因简化牺牲功能 所有原有面板功能正常可用
    控制面分离 UI 作为 Gateway 的控制面，核心体验是助手 用户主要通过渠道与助手交互，UI 用于配置和管理
    多频道统一 单一界面管理所有频道的消息 支持频道切换、统一消息流、频道特定操作
    渐进披露 常用功能首屏可见，高级功能折叠/分组 移动端 5 项核心导航 + 更多抽屉
    场景导向 按使用场景而非功能模块分组 导航按沟通/概览/能力/监控/高级/配置分组
    效率优先 常用操作快捷键 + 快速操作面板 关键操作支持快捷键，Quick Actions 面板可用
    实时反馈 Presence、Typing Indicators 等实时状态显示 用户能感知对方在线状态和输入状态
    双主题一致 浅色/深色下控件行为一致 所有组件在两种主题下样式正常

    2.2 设计原则¶

    1. 信息分层：清晰的视觉层级，重要信息突出
    2. 可访问性：关键操作有 tooltip/aria，颜色对比度符合 WCAG AA
    3. 国际化：所有用户可见字符串走 t()，支持中/英等语言
    4. 响应式：桌面三栏、平板可折叠、移动底部导航 + 抽屉
    5. 一致性：控件样式、交互反馈、动画效果全局统一
    6. 实时优先：所有状态变化即时反映（WebSocket 推送）
    7. 语音一等：语音交互（Voice Wake、Talk Mode）作为一等交互方式
    8. 安全默认：默认隐私安全策略（如 DM Pairing），需显式开放
    9. 渐进增强：核心功能稳定可用，高级功能可配置启用

    3.0.2 实时状态指示¶

    Presence（在线状态）： 

    状态指示器：
🟢 在线（WebSocket 已连接，最近活跃）
🟡 离开（WebSocket 已连接，5 分钟未活跃）
🔴 离线（WebSocket 断开或 30 分钟未活跃）
⚪ 未知（状态信息不可用）

    Typing Indicators（输入指示）： 

    显示时机：
- 对方正在输入时
- 对方正在录制语音时

显示位置：
- 消息气泡下方，或
- 输入框上方

样式：
✏️ 正在输入...
🎤 正在录制...
🔄 Agent 思考中...

def slider_input(
    label: str,
    min_val: float,
    max_val: float,
    value: float,
    step: float = 0.1,
    unit: str = "",
    tooltip: str = "",
    on_change: Callable[[float], Any] | None = None,
) -> ft.Column:
    """滑块 + 数值输入组合控件"""
    from pyclaw.ui.theme import get_theme

    theme = get_theme()

    def _on_slider_change(e: ft.ControlEvent) -> None:
        new_value = float(e.control.value)
        number_field.value = f"{new_value}{unit}"
        number_field.update()
        if on_change:
            on_change(new_value)

    def _on_text_change(e: ft.ControlEvent) -> None:
        try:
            text_val = e.control.value
            # 提取数值部分
            numeric_str = text_val.replace(unit, "").strip()
            new_value = float(numeric_str)
            slider.value = max(min_val, min(max_val, new_value))
            slider.update()
            if on_change:
                on_change(slider.value)
        except ValueError:
            pass

    slider = ft.Slider(
        min=min_val,
        max=max_val,
        value=value,
        divisions=int((max_val - min_val) / step),
        label="{value}%.1f" + unit if unit else "{value}%.1f",
        on_change=_on_slider_change,
        expand=True,
    )

    info_icon = ft.IconButton(
        icon=ft.Icons.INFO_OUTLINE,
        icon_size=16,
        tooltip=tooltip,
        style=ft.ButtonStyle(
            overlay_color=ft.Colors.TRANSPARENT,
            padding=0,
        ),
    )

    header_row = ft.Row(
        controls=[ft.Text(label, size=14), info_icon],
        alignment=ft.MainAxisAlignment.SPACE_BETWEEN,
    )

    number_field = ft.TextField(
        value=f"{value}{unit}",
        width=80,
        dense=True,
        text_align=ft.TextAlign.RIGHT,
        on_change=_on_text_change,
    )

    return ft.Column(
        controls=[
            header_row,
            slider,
            number_field,
        ],
        spacing=8,
        tight=True,
    )

def expandable_section(
    title: str,
    icon: str,
    content: ft.Control,
    initially_expanded: bool = False,
    on_expand: Callable[[bool], Any] | None = None,
) -> ft.Container:
    """可折叠的设置区域"""
    from pyclaw.ui.theme import get_theme

    theme = get_theme()

    expanded_state = {"value": initially_expanded}

    def _toggle(e: ft.ControlEvent) -> None:
        expanded_state["value"] = not expanded_state["value"]
        content_container.visible = expanded_state["value"]
        arrow_icon.icon = (
            ft.Icons.EXPAND_LESS
            if expanded_state["value"]
            else ft.Icons.EXPAND_MORE
        )
        content_container.update()
        arrow_icon.update()
        if on_expand:
            on_expand(expanded_state["value"])

    arrow_icon = ft.Icon(
        ft.Icons.EXPAND_LESS if initially_expanded else ft.Icons.EXPAND_MORE,
        size=20,
        color=theme.colors.muted,
    )

    header = ft.Container(
        content=ft.Row(
            controls=[
                arrow_icon,
                ft.Container(width=8),
                ft.Icon(icon, size=20, color=theme.colors.primary),
                ft.Container(width=8),
                ft.Text(title, size=14, weight=ft.FontWeight.W_500),
            ],
            vertical_alignment=ft.CrossAxisAlignment.CENTER,
        ),
        padding=ft.padding.symmetric(horizontal=16, vertical=12),
        on_click=_toggle,
        border=ft.border.only(
            bottom=ft.BorderSide(0.5, theme.colors.border),
        ),
    )

    content_container = ft.Container(
        content=content,
        padding=ft.padding.all(16),
        visible=initially_expanded,
    )

    return ft.Container(
        content=ft.Column(
            controls=[header, content_container],
            spacing=0,
            tight=True,
        ),
        border_radius=ft.border_radius.all(8),
        bgcolor=theme.colors.surface_container,
    )

def quick_action_card(
    icon: str,
    title: str,
    description: str,
    actions: list[dict[str, Any]],
    on_click: Callable | None = None,
) -> ft.Container:
    """快捷操作卡片"""
    from pyclaw.ui.theme import get_theme

    theme = get_theme()

    action_buttons = []
    for action in actions:
        btn = ft.ElevatedButton(
            action.get("label", ""),
            icon=action.get("icon"),

                color=theme.colors.on_primary_container,

        )

        content=ft.Column(
            controls=[
                ft.Row(
                    controls=[ft.Icon(icon, size=24, color=theme.colors.primary)],
