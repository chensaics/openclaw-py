- **Secrets**: 审计/解析/应用/运行时重载

- **Auto-Reply**: 完整斜杠命令 + 内联指令 + 消息队列 + 调度器
- **Outbound**: 目标解析 + 发送服务 + 消息动作 + 通道适配器 + HTML 导出
