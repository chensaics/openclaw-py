# P1 通道能力 DoD 与验收脚本

## DoD 维度

- 连接稳定性：连接、重连、异常退出恢复
- 消息能力：文本、附件、引用、reaction、typing
- 策略能力：DM/group policy、allowFrom、生效优先级
- 可观测性：错误日志、重试日志、关键指标
- 安全性：来源校验、权限边界、敏感信息脱敏

## 验收脚本建议

- smoke：配置加载 -> 启动 -> 发送 -> 回执
- policy：合法/非法发件人路径验证
- retry：模拟网络失败与死信队列路径
- format：Markdown/纯文本降级一致性
